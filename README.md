## Installation

```bash
$ git clone https://github.com/brightchul/user-service-nestjs.git
$ cd user-service-nestjs
$ npm install
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
